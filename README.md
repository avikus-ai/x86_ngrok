# x86_ngrok

아래 설명은 ngrok을 서비스로 설치하는 방법에대한 것이다. 필요한 설치 방식은 아래의 커멘드를 실행하고 토큰만 있으면 서비스 계정에서 확인가능하다.

## Getting Started
```bash
curl -O https://raw.githubusercontent.com/avikus-ai/x86_ngrok/master/install.sh
chmod +x install.sh
sudo ./install.sh <your_authtoken>
```
